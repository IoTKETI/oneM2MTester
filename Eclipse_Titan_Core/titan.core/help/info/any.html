any

The keyword is used in conjunction with other keywords to refer to at least one entity concerned . It is used...

  1. together with the keywords timer running to check whether a timer is running in the scope unit concerned ;
  2. together with the keyword timeout to wait for the expiry of any timer in the scope unit concerned;
  3. together with the keywords component running to check whether at least one parallel component is still running;
  4. together with the keyword done to check whether at least one parallel component has finished execution.
  5. together with the keyword port to receive messages from any port.

Related keywords:

  • component
  • done
  • receive
  • running
  • timeout
  • timer
