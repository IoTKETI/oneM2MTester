goto

The keyword performs a jump in the program execution to a specific place marked by a label.

  • Forward references (i.e., using a yet undeclared label in a goto statement) are allowed.

Related keyword:

  • label

goto label_identifier;

  • label_identifier is the name used to refer to the label where goto jumps to. It must begin with a letter, may contain letters, numbers and underscore characters.

Example 1:

goto Bilabiaalis;

A jump is made to the label called Bilabiaalis.
