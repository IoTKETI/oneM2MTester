General Directions

Rule-Names

  • Clicking on the rule-name will move you to the list of references to the selected rule-name.
  • Rule-names which are shown in red cannot be selected because they are never referenced in the specification. If a rule-name other than the root rule-name is shown in red, the specification is not in error, but there are unreferenced (redundant) productions in the specification.

References

  • Clicking on references in the production will move you to the production with the rule-name selected.
  • References which are shown in red cannot be selected because they are never defined in the specification. If there are references shown in red then the specification is either incomplete or in error.
