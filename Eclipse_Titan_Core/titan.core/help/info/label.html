label

The keyword marks a specific place in test cases, functions, altsteps and the control part of a module.

  • The keyword goto is used to jump in the program execution to a label.
  • Labels must not be defined as first statement of an alternative or top alternative inside an alt statement, interleave statement or altstep.

Related keyword:

  • goto

label label_identifier;

  • label_identifier is the name used to refer to the label. It must begin with a letter, may contain letters, numbers and underscore characters.

Example 1:

label Labiaalis;

The label defined is called Labiaalis.
